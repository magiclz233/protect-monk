# 九章主题棋盘与关卡机制设计

> 版本：1.0 | 日期：2026-07-08
> 用途：为九章每关设计独立棋盘配色、锁定格障碍物、关卡特殊机制，全部拟合西游记原著场景

## 一、设计理念

### 1.1 目标

当前所有关卡共用一套棋盘配色和锁定格样式，缺乏章节辨识度。改为每章一章一景，棋盘色彩、障碍物造型、特殊机制全部拟合《西游记》原著对该地点的环境描写。

### 1.2 三合一设计

每章设计三个维度联动：

| 维度 | 说明 | 影响 |
|------|------|------|
| **棋盘配色** | 面板底色、可布阵格、路径格、锁定格的颜色 | 视觉氛围 |
| **障碍物** | 锁定格的造型，需 AI 生成 80×80 PNG | 场景叙事 |
| **特殊机制** | 部分关卡的额外规则（持续伤害、减速、复活等） | 玩法深度 |

### 1.3 棋盘配色自动推导规则

设计文档 §3.1 已定义了每章的 `战场底色` 和 `路径色`，此处基于这两个色推导完整棋盘：

```
面板填充 = 玄青 #1a1814（统一不变，保证全局一致性）
面板边框 = 章节主题色，降低 alpha 至 42%
可布阵格 = 战场底色，alpha 90%
可布阵边框 = 路径色 提亮 15%，alpha 60%
路径格 = 路径色 加深 20%，alpha 90%
路径格边框 = 朱砂 #c43d30，alpha 50%（统一，代表敌路危险）
锁定格 = 路径色 加深 30% + 去饱和 20%，alpha 98%
锁定格边框 = 章节主题色，alpha 65%
路径宽线 = 绛红 #8b1a1a 暖化，alpha 84%
路径中线 = 章节暗部色 暖化，alpha 76%
路径金线 = 鎏金 #f0c15a，alpha 88%（统一不变）
```

### 第 1 章 — 五行山「悟空破山」

**原著场景**：如来将孙悟空压在五行山下五百年，山顶贴有金字封条。唐僧揭去封条，悟空裂山而出。

| 棋盘配色 | 值 | 说明 |
|----------|-----|------|
| 面板边框 | `#5b8c5a` α42% | 山绿色 |
| 可布阵格 | `#1e2a1c` α90% | 深林山地色 |
| 可布阵边框 | `#e0c8a0` α60% | 沙色提亮 |
| 路径格 | `#b89068` α90% | 沙色山路 |
| 路径格边框 | `#c43d30` α50% | 朱砂敌路 |
| 锁定格 | `#2a3824` α98% | 暗青山岩 |

**解锁动画**：封条先燃烧化为金粉 → 石体裂开崩碎 → 金光消散。

| 机制 | 说明 |
|------|------|
| 类型 | 教学关（新手友好） |
| 特殊规则 | 无特殊机制，标准的塔防教学流程 |

## 五、AI 生成素材清单

### 5.1 建立风格（每条提示词前附上）

```
我在为一款《守护唐僧》Q版西游合成塔防微信小游戏生成可商用2D游戏素材。

美术风格：扁平国潮风（Guochao Flat）—— 扁平矢量 + 中式纹样点缀 + 2层cel明暗 + 3-4px粗轮廓线 + 高饱和配色。
光影：单层cel shading，亮面100%+暗面75%，不要渐变过渡，不要厚涂。
线条：粗轮廓描边，比填充色深2档，小尺寸下轮廓必须清晰可读。
视角：轻微45度俯视棋盘视角，适合80×80棋盘格。

本批素材是棋盘上的障碍物（locked cell），每个80×80，不是角色。透明背景PNG，真alpha通道，无文字无水印，无灰白棋盘格。
绝对禁止：写实/3D/厚涂/渐变/像素风/真实照片质感/欧美魔幻/日本动画风。
```

### 5.2 障碍物静态图（9 张）

> 生成一张横向 spritesheet，9 列 × 1 行，每格 80×80，整张 720×80。9 个不同的 Q版扁平国潮风棋盘障碍物，从左到右依次：
>
> **第1格（五行山·符印封石）**：圆润山石，石绿#2ea07a主色翠绿#5bc48a亮面，表面贴一张金色#f0c15a符印封条，封条上有简化梵文咒纹符号，封条微微发光。粗墨线描边。
>
