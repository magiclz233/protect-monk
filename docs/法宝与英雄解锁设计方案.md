## 一、设计目标

当前版本存在三个结构性问题：

| 问题 | 现状 | 目标 |
|------|------|------|
| 道具混在抽卡池 | 抽 5 张只有 3-4 张是小兵/碎片 | 抽卡只出小兵和碎片，法宝走独立系统 |
| 英雄获取无节奏 | 14 个英雄全在池里，新手抽到后期碎片用不了 | 初始 5 个 + 章节按剧情解锁 9 个 |
| 道具使用无策略 | 随机出，急需时没有，溢出时堆着 | 战前选择携带，局内 CD 手动释放 |

核心原则：

> **抽卡只管合成，法宝只管战术。两个系统彻底解耦，各自做到最纯粹。**

## 二、抽卡池简化

### 2.1 最终卡池

```
┌──────────────────────────────┐
│                              │
│   小兵 (R1)       ~66%       │
│   英雄碎片         ~22%       │
│   开山斧           ~12%       │
│                              │
└──────────────────────────────┘
```

> 道具全部从抽卡池移除，走独立的法宝系统。开山斧保留在抽卡池，同时法宝 CD 提供保底。

### 2.2 旧 vs 新对比

| 项目 | 旧 | 新 |
|------|-----|-----|
| 小兵卡 | 70% | 66% |
| 英雄碎片 | 18% | 22% |
| 开山斧 | 4.8% | **12%** |
| 其他道具 | 7.2% | **0%（移除）** |

### 2.3 开山斧双来源

开山斧同时存在于抽卡池和法宝系统：

| 来源 | 概率 / CD | 定位 |
|------|----------|------|
| 抽卡 | 12%/卡 | 主要获取，爆发惊喜 |
| 法宝 CD | 60s/把 | 保底兜底，运气差也不卡死 |

| 场景 | 抽卡产出 | CD 产出 | 合计 |
|------|---------|---------|------|
| Defense 18min（10 次召唤） | ~6 把 | 18 把 | **~24 把** |
| Journey 3min（2 次召唤） | ~1.2 把 | 3 把 | **~4 把/关** |

> Defense 全图最大需求约 25 把（Ch9 灵山），24 把刚好覆盖。Journey 每关需求 4-5 格，4 把刚好够用。

## 三、法宝系统

### 3.1 架构

```
战前选择携带 → 局内独立 CD → 手动释放
```

- 法宝在进入关卡前从已解锁池中选择携带
- 每个法宝有独立的 CD 计时器
- 局内 CD 冷却完成后手动点击释放
- 法宝可消耗**灵蕴**升级（Lv1 → Lv2 → Lv3），灵蕴来自通关结算，详见[货币系统](#七货币与段位系统)

### 3.2 初始法宝（3 件，开局即拥有）

对应唐僧出发前已有的基础保障：

| # | 法宝 | 西游出处 | CD | Lv1 效果 | Lv2 效果 | Lv3 效果 |
|---|------|---------|-----|----------|----------|----------|
| 1 | **开山斧** | 樵夫赠斧开路 | 60s | 解锁 1 格 | CD 50s | 一次解锁 **2 格** |
| 2 | **回山符** | 观音所赐护身符 | 100s | 弹回 2 次 | 弹回 3 次 | 弹回 **4 次** |
| 3 | **杨枝甘露** | 观音净瓶甘露 | 45s | 全体回 20% HP | 全体回 25% HP | 全体回 30% HP + 净化 |

**为什么是这三件：**

| 法宝 | 解决的问题 |
|------|-----------|
| 开山斧 | 棋盘扩展是核心玩法，不能卡在抽卡里 |
| 回山符 | 3 滴血前期太脆，必须给唐僧一次容错 |
| 杨枝甘露 | 友方受击实装后，必须能救小兵/英雄 |

> 这三件覆盖了「开格子 + 保唐僧 + 保友方」的基础循环，是体验底线而非策略选择。

### 3.4 唐僧诵经光环（被动）

唐僧不再是纯粹的血条，而是提供光环的核心单位：

| HP 状态 | 覆盖范围 | 攻击光环 | 回血光环 |
|----|---------|---------|---------|
| ≥67% | 全图 | +10% ATK | 按缺血压力动态回血，基础每秒 0.3% 最大 HP |
| 34%-66% | 全图 | +5% ATK | 按缺血压力动态回血，最高每秒 3% 最大 HP |
| <34% | 全图 | 无攻击加成 | 仍保留全图回血，帮助残血防线稳住 |

当前代码公式为：`min(3%, 0.3% + 缺血比例 × 1.8% + 缺血比例 × 最大血量压力 × 1.1%)`，其中最大血量压力按唐僧最大血量从 3 到 7 线性提高。

> 诵经光环覆盖全图，不再要求友方围绕唐僧布阵。唐僧血量影响光环强度：血量越低，攻击加成衰减；缺血压力越高，回血越积极。
> 锦斓袈裟使用时，光环效果翻倍/三倍，是终极爆发时机。

### 3.5 法宝升级系统

消耗**灵蕴**在法宝选择界面升级：

| 等级 | 消耗灵蕴 | 累计 | 通用强化 |
|------|---------|------|---------|
| Lv1 → Lv2 | 80 ✦ | 80 ✦ | CD -10% 或效果 +25% |
| Lv2 → Lv3 | 200 ✦ | 280 ✦ | CD -15% 或效果 +50% |

> 全部 12 件法宝升满需要 12 × 280 = **3360 ✦**，是长线养成目标。
> 灵蕴获取方式见[货币系统](#七货币与段位系统)。

## 四、英雄解锁系统

### 4.1 设计原则

> **碎片池随章节逐步扩展。解锁后才进池，碎片即抽即用。**

- 初始 5 个英雄的碎片可抽取
- 每通一章，1 个新英雄加入抽卡池
- 通完 Ch9，全部 14 个英雄进池

### 4.2 初始英雄（5 个，开局即可抽取）

| 英雄 | 稀有度 | 定位 | 为什么是初始 |
|------|--------|------|------------|
| 猪八戒 | 普通 | 近战肉盾 | 取经二师兄，前期必须有一个坦扛伤害 |
| 白龙马 | 普通 | 远程穿透 | 取经坐骑，前期远程输出 |
| 太上老君 | 核心 | 持续输出 | 灼烧 DOT，天庭炼丹师 |
| 黑熊精 | 普通 | 物理肉盾 | 还不到 Ch1 就遇到的妖怪 |
| 托塔天王 | 普通 | 群体控制 | 天庭将领，15% 定身 |

> 两个坦 + 两个远程 + 一个控制，新手池有完整的战术搭配。

### 4.5 选中英雄权重

抽卡时，已选中的上阵英雄碎片权重 **5×**，未选中英雄权重 1×。当前每关最多可出战 4 名英雄，下表按“选中 3 名核心培养对象”测算碎片定向效率。

| 场景 | 池大小 | 测算选中人数 | 特定英雄碎片/5 抽 |
|------|--------|----------|------------------|
| 前期（Ch0） | 5 个 | 3/5 | 22% × (5/17) ≈ **0.65 个** |
| 中期（Ch5） | 10 个 | 3/10 | 22% × (5/29) ≈ **0.38 个** |
| 后期（Ch9） | 14 个 | 3/14 | 22% × (5/37) ≈ **0.30 个** |

> 前期定向养成快，后期变慢但池子大了玩法多。激活核心英雄（3 碎片）前期约 5 次召唤，后期约 10 次召唤。

## 六、完整游戏循环

### 6.1 局外循环

```
通关章节
    │
    ├──→ 解锁新英雄（碎片加入抽卡池）
    ├──→ 解锁新法宝（加入法宝选择池）
    ├──→ 获得灵蕴（结算产出）
    │       │
    │       └──→ 升级法宝
    │
    └──→ 开启下一章（重复）
```

### 6.2 局内循环

```
开始关卡
    │
    ├──→ 战前选择携带法宝（3~6 个）
    │
    ├──→ 开局：有初始 50 🍑 + 6-7 个开放格
    │
    ├──→ 战斗中：
    │      ├── 击杀敌人获得仙桃（局内货币，每局重置）
    │      ├── 抽卡消费仙桃 → 小兵 + 碎片 + 开山斧
    │      ├── 合成小兵进阶
    │      ├── 碎片激活英雄
    │      ├── 开山斧开格子
    │      └── 法宝 CD 好了手动释放
    │
    └──→ 通关 → 结算灵蕴；失败 → 少量灵蕴
```

### 6.3 法宝使用节奏（Defense 模式示例）

```
Wave 1-4    │ 不开法宝，攒 CD
Wave 5      │ 精英出现 → 照妖镜
Wave 6-9    │ 怪群压力 → 紧箍咒
Wave 10     │ Boss 黑熊精 → 避火罩保前排 + 杨枝甘露抬血
Wave 11-14  │ 连续怪群 → 芭蕉扇推一波
Wave 15     │ 漏怪 → 回山符弹回
```

## 七、货币与段位系统

### 7.1 三种货币

| | 仙桃 🍑 | 灵蕴 ✦ | 段位 🎖 |
|------|---------|---------|---------|
| **来源** | 局内击杀敌人 | 通关结算 | Defense 模式最高波次 |
| **周期** | 每局重置 | 永久累积 | 取最高记录 |
| **用途** | 局内召唤抽卡 | 升级法宝 | 排行榜展示 / 头像框 |
| **适用模式** | Journey + Defense | Journey 为主 | Defense 专属 |

> 仙桃只存在于局内，只用于召唤。灵蕴是跨局养成货币。段位是竞技荣誉。
> 三者完全独立，用来源/用途/命名三重区分。

### 7.2 灵蕴产出公式

```
灵蕴 = 基础 + 星级加成 + Boss 加成

基础 = 关卡编号 × 2
星级：一星 +0 / 二星 +20 / 三星 +50
Boss 关（关卡编号 % 9 == 0）额外：+30
```

| 关卡 | 一星 | 二星 | 三星 |
|------|------|------|------|
| Lv1 (小关) | 2 | 22 | 52 |
| Lv9 (Boss 关) | 48 | 68 | **128** |
| Lv27 (Boss 关) | 84 | 104 | 164 |
| Lv45 (Boss 关) | 120 | 140 | 200 |
| Lv81 (终关) | 192 | 212 | **272** |

> 三星通关 Boss 关是大额灵蕴来源，激励玩家追求三星 + 尽快推到 Boss 章。

### 7.3 段位系统

基于 Defense 模式**最高波次**，设 10 个段位：

| 段位 | 最高波次 | 视觉标识 | 说明 |
|------|---------|---------|------|
| **凡修** | 0-4 | 无框 | 初入守护 |
| **天兵** | 5-9 | 铁框 ⚪ | 小有历练 |
| **天将** | 10-14 | 铜框 🟤 | 独当一面 |
| **星君** | 15-19 | 银框 ⚪ | 星辰之力 |
| **散仙** | 20-24 | 金框 🟡 | 超越凡尘 |
| **真君** | 25-29 | 紫金框 🟣 | 一方霸主 |
| **金仙** | 30-34 | 赤金框 🔴 | 金身不坏 |

### 7.4 排行榜

好友榜 + 全服榜，三级排序：

| 排序优先级 | 指标 | 说明 |
|-----------|------|------|
| 1 | **最高波次** | Defense 模式核心指标 |
| 2 | **最高击杀** | 那局的杀敌数 |
| 3 | 达成时间 | 先达成者排前 |

> 段位完全由最高波次决定，跟章节进度无关，跟抽卡运气无关。纯竞技实力。
